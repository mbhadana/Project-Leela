# Leela V1 — Portable Edition

## Quick Start

1. **Extract** this folder anywhere on your computer
2. **Run** `LeelaV1.exe`
3. **Enter your API key** when prompted on first launch

That's it. Leela is ready.

---

## Getting Your API Key

1. Visit [sarvam.ai](https://www.sarvam.ai/)
2. Create a free account
3. Go to **Dashboard → API Keys**
4. Copy your key (starts with `sk_`)
5. Paste it into Leela's setup screen

---

## Your Privacy

🔒 **Your API key stays on your machine.**

- Your key is encrypted using your operating system's secure storage
- No data is ever sent to the developer
- Communication goes directly: **Your computer → Sarvam AI**
- The developer does not have access to your key
- No tracking, no analytics, no telemetry

---

## How It Works

- **Ctrl+Space** - Start/stop voice dictation
- **Select text + Ctrl+Space (tap)** - Instantly polish selected text
- **Select text + Ctrl+Space (hold + speak)** - Voice command on selected text
- **Alt+Space** - Toggle AI assistant side panel
- **System tray icon** - Right-click for dashboard and settings

## Requirements
